# Refurbished Mobile Marketplace

## Project Overview
This project is a refurbished mobile marketplace where users can buy and sell refurbished mobile devices. Our goal is to provide a sustainable and cost-effective platform for quality mobile phones that have been refurbished and certified for resale.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Hosting:** AWS, Heroku
- **Version Control:** Git
- **Continuous Integration/Deployment:** GitHub Actions

## Features
- User authentication (sign up, login, logout)
- Browse and search for refurbished mobiles
- Product details page with images and specifications
- User reviews and ratings for products
- Seller dashboard to manage listings
- Secure payments and order tracking

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/shiv858/refurb-mobile-store.git
   cd refurb-mobile-store
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open your browser and navigate to `http://localhost:3000` to see the app in action.
